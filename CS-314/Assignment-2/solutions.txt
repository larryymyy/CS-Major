Part 4
======
Hexdecimal -> Decimal

a. 0x000000C6
	->    0   0    0   0    0   0    C   6
	-> 00000000 00000000 00000000 11000110
	-> 2^7 + 2^6 + 2^2 + 2^1
	-> 128 + 64 + 4 + 2
	-> 200


b. 0xFFFFFEC6
	->    F   F    F   F    F   E    C   6
	-> 11111111 11111111 11111110 11000110
	-> -2^31 + 2^30 + 2^29 + ... + 2^9 + 2^7 + 2^6 + 2^2 + 2^1
	-> -2147483648 + 1073741824 + 536870912 + 268435456 + 134217728    [WRAP]
		+ 67108864 + 33554432 + 16777216 + 8388608 + 4194304 + 2097152 [WRAP]
		+ 1048576 + 524288 + 262144 + 131072 + 65526 + 32768 + 16384   [WRAP]
		+ 8192 + 4096 + 2048 + 1024 + 512 + 128 + 64 + 4 + 2
	-> -314


c. 0xFFFFFFFF
	->    F   F    F   F    F   F    F   F
	-> 11111111 11111111 11111111 11111111
	-> -2^31 + 2^30 + 2^29 + ... + 2^1 + 2^0
	-> -1



Part 5
======
Hexadecimal -> IEEE 754 Single-Precision Floating-Point

a. 0x80000000
	->    8   0    0   0    0   0    0   0
	-> 10000000 00000000 00000000 00000000
	-> sign = -
	-> exponent = 127
	-> fraction = 1.0
	-> fraction*2^exp
	-> 0000...0000
	-> -0.0


b. 0x439D0000
	->    4   3    9   D    0   0    0   0
	-> 01000011 10011101 00000000 00000000
	-> sign = +
	-> exponent = 8
	-> fraction = 1.00111010000000000000000
	-> fraction*2^exponent
	-> 100111010.0000...0000
	-> 314.0


c. 0xC1340000
	->    C   1    3   4    0   0    0   0
	-> 11000001 00110100 00000000 00000000
	-> sign = -
	-> exponent = 3
	-> fraction = 1.01101000000000000000000
	-> fraction*2^exponent
	-> 1011.0100...0000
	-> -11.25



Part 6
======
Decimal -> Hex via IEEE 754 Single-Precision Floating-Point Arithmetic
a. -1.0
	-> binary of 1 -> 1
	-> sign = -
	-> exponent = 0 => 01111111
	-> 1 01111111 00000000000000000000000
	-> 10111111 10000000 00000000 00000000
	-> 1011 1111 1000 0000 0000 0000 0000 0000
	->    B    F    8    0    0    0    0    0
	-> 0xBF800000

b. 16.75
	-> binary of 16 -> 10000
	-> binary of .75 -> 0.11
	-> sign = +
	-> exponent = 4 => 10000011
	-> 0 10000011 00001100000000000000000
	-> 01000001 10000110 00000000 00000000
	-> 0100 0001 1000 0110 0000 0000 0000 0000
	->    4    1    8    6    0    0    0    0
	-> 0x41860000

c. -314.0
	-> binary of 314 -> 100111010
	-> exponent = 8 => 10000111
	-> sign = -
	-> 1 10000111 00111010000000000000000
	-> 11000011 10011101 00000000 00000000
	-> 1100 0011 1001 1101 0000 0000 0000 0000
	->    C    3    9    D    0    0    0    0
	-> 0xC39D0000
