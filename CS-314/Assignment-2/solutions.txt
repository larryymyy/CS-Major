Part 4

Hexdecimal -> Decimal

a. 0x000000C6
	->    0   0    0   0    0   0    C   6
	-> 00000000 00000000 00000000 11000110
	-> 2^7 + 2^6 + 2^2 + 2^1
	-> 128 + 64 + 4 + 2
	-> 200


b. 0xFFFFFEC6
	->    F   F    F   F    F   E    C   6
	-> 11111111 11111111 11111110 11000110
	-> -2^31 + 2^30 + 2^29 + ... + 2^9 + 2^7 + 2^6 + 2^2 + 2^1
	-> -2147483648 + 1073741824 + 536870912 + 268435456 + 134217728    [WRAP]
		+ 67108864 + 33554432 + 16777216 + 8388608 + 4194304 + 2097152 [WRAP]
		+ 1048576 + 524288 + 262144 + 131072 + 65526 + 32768 + 16384   [WRAP]
		+ 8192 + 4096 + 2048 + 1024 + 512 + 128 + 64 + 4 + 2
	-> -314


c. 0xFFFFFFFF
	->    F   F    F   F    F   F    F   F
	-> 11111111 11111111 11111111 11111111
	-> -2^31 + 2^30 + 2^29 + ... + 2^1 + 2^0
	-> -1
